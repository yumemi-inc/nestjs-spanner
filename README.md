## Description

Thin ORM for Spanner depend on [Nest](https://github.com/nestjs/nest) framework by TypeScript.

## Installation

```bash
$ yarn 
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
